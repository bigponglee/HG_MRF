# HG_MRF
A simple implementation of the HG-MRF for MRF reconstruction: "HG-MRF: Advanced Brain MRF reconstruction using Anatomy-Aware Hierarchical Hypergraph".
# Files
- `data`: A folder containing the input data used in the main program.
- `hg_func`: A folder containing functions used to create and manipulate hypergraphs.
- `py_func`: A folder containing Python functions used in the main program.
- `utils`: A folder containing utility functions used in the main program.
- `main_vds_spiral_hg`: The main program for running the HG-MRF algorithm.
